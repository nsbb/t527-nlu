## 증강 전략

65개 조합 × 각 300개 = **19,500개 목표**

시나리오 많은 조합 → 시나리오당 적게 생성:
- `부가|날씨|조회` (19개) → 시나리오당 ~16개
- `부가|뉴스|조회` (16개) → 시나리오당 ~19개

시나리오 적은 조합 → 시나리오당 많이 생성 (어미변형+간접+STT오류):
- `더보기|엘리베이터|호출` (1개) → 300개 변형
- `메인|시간|조회` (1개) → 300개 변형
