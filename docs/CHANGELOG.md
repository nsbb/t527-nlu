| 멀티턴 온도 조정 "2도 낮춰줘" after "24도로" → "22도에서 22도로" (WRONG) | 조정 전 온도(24도) 정보가 response generator에 전달되지 않고, hardcoded 22도 사용 | DST → Pipeline → ResponseGen 전체에 `old_value` 전달 체인 추가 |
